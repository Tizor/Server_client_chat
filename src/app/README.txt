- Вторая версия чата -
Теперь в общении могут принимать участие несколько человек.
Однако тут присутсвует сложность в постоянном наборе имени адресата,
кому нужно отправить сообщение.
Можно исправить в следующих версиях программы.


Чтобы отправить сообщение от любого клиента, введите сообщение, затем знак «#» и имя получателя. Обратите внимание, что эта реализация дает имена как «клиент 0», «клиент 1 ″….» Клиент n », и поэтому имена должны быть добавлены в конец. После этого нажмите клавишу Enter.
После отправки сообщения обработчик этого клиента получит сообщение, и оно будет доставлено указанному клиенту.
Если какой-либо клиент отправляет сообщение этому клиенту, поток readMessage автоматически распечатает сообщение на консоли.
После того, как клиент закончил общение, он может отправить сообщение о выходе без имени получателя, чтобы сервер знал, что этот клиент вышел из системы.
Чтобы избежать ошибок, перед закрытием терминала рекомендуется отправить клиенту сообщение о выходе из системы.

From client 0 :

hello#client 1
client 1 : heya
how are you#client 1
client 1 : fine..how about you
logout

From client 1 :

client 0 : hello
heya#client 0
client 0 : how are you
fine..how about you#client 0
logout
